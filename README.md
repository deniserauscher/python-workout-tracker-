# Python Workout Tracker
## Overview

This project is a simple command-line workout tracker written in Python.
It allows users to log workouts, store them locally, and analyze recent activity such as total training time and calories burned.

The project demonstrates basic concepts such as:

* Object-oriented programming (OOP)
* File handling
* User input processing
* Simple data analysis

## Features

* Add workouts (running, swimming, pilates)
* Automatically save workouts to a file
* Load previous workout history on startup
* Display all logged workouts
* Calculate total training time (last 7 days)
* Estimate calories burned (last 7 days)


## Technologies

* Python 
* File-based storage (`.txt` file)

##  Usage

1. Clone the repository
```bash
git clone https://github.com/your-username/workout-tracker.git
```
2. Navigate into the project
```bash
cd workout-tracker
```
 3. Run the program
```bash
python main.py
```
## Menu Options
After running the script, you can choose:

```
1 - Add workout
2 - Show workouts
3 - Show total time (last 7 days)
4 - Show calories burned (last 7 days)
```
